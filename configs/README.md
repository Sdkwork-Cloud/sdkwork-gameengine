# Runtime Config Templates

Safe checked-in examples for `sdkwork-games`. Local overrides such as `.env.local` and `*.local.toml` must stay gitignored.

| File | Purpose |
| --- | --- |
| `games.database.example.toml` | PostgreSQL connection template for `GAMES_DATABASE_*` |
| `topology/` | Deployment profile and service-layout env templates |

See `../sdkwork-specs/CONFIG_SPEC.md` and `../sdkwork-specs/ENVIRONMENT_SPEC.md`.
